The protocol demo is devnet-native and does not depend on the OWS gateway example. It exercises the settlement primitives directly against the live V4 deployment.

## Run

```bash
cd /home/heis/agon/agon-protocol
npm run demo:agon-protocol
npm run benchmarks:generate
```

The first command executes the live scenarios. The second turns the last-run manifest plus the synthetic sizing scripts into committed benchmark snapshots and generated benchmark pages for this documentation site.

## What The Demo Covers

1. latest-commitment settlement
2. bundle settlement
3. single-payer cooperative clearing
4. bilateral cooperative clearing
5. the largest executable multilateral round found on the current devnet deployment
6. capacity summary and artifact output

The run writes a machine-readable manifest to `config/agon-protocol-demo-last-run.json`. That manifest is the source for the live benchmark export pipeline.
