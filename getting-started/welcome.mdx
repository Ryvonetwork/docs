Agon Protocol is a **non-custodial clearing protocol for machine payments** on Solana.

It is built for repeated relationships: agents paying APIs, inference endpoints, data feeds, marketplaces, and other agents. Instead of settling every small interaction as a standalone transfer, Agon lets participants exchange cumulative offchain commitments and settle only the latest state that matters.

## What Agon Does

Agon keeps balances in protocol state, uses permanent participant identities and permanent directed lanes, and supports three settlement modes:

1. latest commitment settlement for one lane
2. bundle settlement for many lanes to one payee
3. cooperative clearing rounds that advance many lanes together and apply only the residual balance changes that remain after obligations are netted

That makes Agon a better fit for machine payments than exact per-payment settlement when the same parties interact repeatedly.
