The protocol demo in the protocol repo is the main end-to-end reference run for the live V4 deployment.

## Run

```bash
cd /home/heis/agon/agon-protocol
npm run demo:agon-protocol
npm run benchmarks:generate
```

The first command runs the live settlement scenarios. The second turns the last-run manifest plus the synthetic sizing scripts into committed JSON snapshots under `benchmarks/` and refreshes the generated benchmark pages in this docs site.

This keeps the public docs tied to live protocol evidence instead of hand-maintained benchmark copy.
