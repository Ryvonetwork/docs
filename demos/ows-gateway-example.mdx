---
title: "OWS Gateway Example"
description: "An application-layer example of Agon backing paid RPC access."
---

The OWS gateway demo shows one application-layer use of Agon: metered Solana RPC access with OWS-signed payment updates.

It is useful if you want to see how the protocol can back x402-style API monetization and repeated paid requests. It is not required for understanding the protocol itself. The protocol-only devnet demo remains the best reference for V4 settlement behavior and benchmarks.

At a higher level, this is also one example of the broader [operator hub model](/demos/operator-hub-model): users pay one operator, and that operator coordinates paid access to downstream infrastructure on top of Agon.
