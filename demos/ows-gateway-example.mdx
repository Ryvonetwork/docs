The OWS gateway demo is still useful, but it now sits as an **application example** rather than the main protocol narrative.

It demonstrates:

- OWS-signed payment updates
- an RPC-provider use case
- how Agon can back x402-style API monetization

For Colosseum, the core story should remain protocol-first:

- latest commitment compression
- bundle settlement
- cooperative clearing
- benchmarked settlement density
