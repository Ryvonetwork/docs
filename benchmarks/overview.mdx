Agon uses two benchmark families:

## 1. Live Devnet Benchmarks

These come from the protocol-only devnet demo and capture:

- the actual scenario shape
- actual explorer links
- actual serialized transaction bytes where measured
- settlement compression against an exact-payment baseline

## 2. Synthetic Capacity Benchmarks

These come from deterministic sizing scripts and capture:

- unilateral message density
- bundle settlement limits
- cooperative clearing-round headroom
- hypothetical BLS headroom

The generated pages in this section are produced from committed JSON snapshots rather than hand-maintained prose.
